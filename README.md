open project.greenfoot to launch game

### Controls
Player 1 Movement
```
A - Left
D - Right
W - Jump
Q - Punch
E - Kick
F - shoot
```

Player 2 Movement 
```
Left Arrow - Left
Right Arrow - Right
Up Arrow - Jump
, - Punch
. - Kick
/ - shoot
```
