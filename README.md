# READ ME 
*ONLY use on Greenfoot 2.4.0 (Any newer, or older versions may not work)

## Launching
open project.greenfoot to launch game
see game_screenshot.png for a preview of the game

## Controls
### Player 1 Movement
A - Left

D - Right

W - Jump

Q - Punch

E - Kick

F - shoot

### Player 2 Movement 
Left Arrow - Left

Right Arrow - Right

Up Arrow - Jump

, - Punch

. - Kick

/ - shoot
